Docker Engine 概述

Docker Engine 是一个开源的容器化技术，用于构建和容器化应用程序。它作为一个客户端-服务器应用程序，包括：

  • 一个长时间运行的守护进程 (dockerd)
  • 用于与 Docker 守护进程通信的 API
  • 一个命令行界面 (CLI) 客户端 (docker)

CLI 使用 Docker API 控制或与 Docker 守护进程交互。守护进程创建和管理 Docker 对象，如镜像、容器、网络和卷。

更多信息请参考 Docker Engine Overview。
